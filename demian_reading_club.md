# 🌟 데미안 독서모임
> *"새는 알에서 나오려고 투쟁한다. 알은 세계다. 태어나려는 자는 하나의 세계를 파괴해야 한다."*
> — 헤르만 헤세, 《데미안》

---

## 📖 우리의 모임에 대하여

이곳은 헤르만 헤세의 《데미안》을 함께 읽고,  
싱클레어처럼 자기 자신을 향한 여정을 나누는 공간입니다.

우리는 각자의 알을 깨고 있는 중입니다. 🥚✨

---

## 🦅 아브락사스 — 우리의 상징

> 아브락사스(Abraxas) — 신과 악마, 빛과 어둠을 동시에 품은 존재.  
> 선과 악을 넘어, 자기 자신이 되는 것.

| 상징 | 의미 |
|------|------|
| 🐦 새 | 자아의 각성, 세계를 깨고 나옴 |
| 🌑 두 세계 | 밝은 세계 vs 어두운 세계, 내면의 분열 |
| 🔥 카인의 표식 | 선택받은 자, 다름을 두려워하지 않는 용기 |
| 🌀 아브락사스 | 선악을 초월한 완전한 존재 |

## 📅 모임 일정

| 회차 | 날짜 | 범위 | 키워드 |
|------|------|------|--------|
| 1회 | | 1장 — 두 세계 | 싱클레어의 어린 시절, 밝은 세계와 어두운 세계 |
| 2회 | | 2장 — 카인 | 데미안과의 만남, 카인의 표식 |
| 3회 | | 3~4장 — 베아트리체 | 사랑과 이상, 꿈속의 여인 |
| 4회 | | 5~6장 — 새는 알에서 나온다 | 피스토리우스, 아브락사스 |
| 5회 | | 7~8장 — 에바 부인 | 에바 부인과의 만남, 자아의 완성 |
| 6회 | | 전체 리뷰 | 나의 데미안, 나만의 해석 |

---

## 💬 매회 나눌 질문들

### 🌑 첫 번째 — 두 세계
- 내 삶에서 "밝은 세계"와 "어두운 세계"는 어디인가?
- 나는 지금 어느 세계에 살고 있다고 느끼는가?

### 🔖 두 번째 — 카인의 표식
- 나는 주변과 다르다고 느낀 적이 있는가? 그 순간은?
- 데미안처럼 나에게 영향을 준 사람이 있는가?

### 💭 세 번째 — 자아의 탐색
- 내 안의 "아브락사스"는 어떤 모습인가?
- 나는 지금 어떤 알 속에 있는가?

### ✨ 마지막 — 나의 데미안
- 이 책이 지금의 내게 건네는 메시지는?
- 나는 어떤 세계를 파괴하고 태어나려 하는가?

> "우리가 어떤 사람을 미워할 때, 우리는 그의 형상 속에서 우리 자신 안에 깃들어 있는 무언가를 미워하는 것이다."

> "사랑은 구걸해서는 안 되고 요구해서도 안 된다. 사랑은 스스로 확신을 가질 힘을 갖추어야 한다."

> "꿈은 거짓말을 하지 않는다."
