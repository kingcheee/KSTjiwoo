### 3. 핵심 개념 및 코드/공식 정리

**1. `matplotlib.pyplot` 라이브러리**
*   **개념:** 파이썬에서 2D 그래프를 그리기 위한 표준 라이브러리입니다. 과학 계산, 통계 분석, 데이터 시각화 등 다양한 분야에서 활용됩니다. `pyplot` 모듈은 MATLAB과 유사한 인터페이스를 제공하여 쉽게 그래프를 생성할 수 있도록 돕습니다.
*   **사용 예:** `import matplotlib.pyplot as plt`

**2. `show_history` 함수**
*   **목적:** 딥러닝 모델의 훈련(train) 및 검증(validation) 과정에서 기록된 손실(loss)과 정확도(accuracy) 이력을 시각화하여 그래프로 보여줍니다. 이를 통해 모델의 학습 진행 상황, 과적합(overfitting) 여부, 그리고 일반화 성능 등을 직관적으로 파악할 수 있습니다.
*   **파라미터:**
    *   `history`: 딥러닝 모델의 학습 이력을 담고 있는 딕셔너리입니다. 일반적으로 `{'train_loss': [...], 'train_acc': [...], 'val_loss': [...], 'val_acc': [...]}`와 같은 형태를 가집니다.
    *   `metric`: 시각화할 지표를 지정하는 문자열입니다. `'loss'` 또는 `'acc'`(기본값) 중 하나를 받습니다.
*   **주요 기능:**
    *   `metric` 값에 따라 플로팅할 데이터의 키(`train_metric_name`, `val_metric_name`)를 결정합니다.
    *   `plt.plot()`을 사용하여 훈련 데이터와 검증 데이터를 각각 다른 색상과 라벨로 그래프에 그립니다.
    *   `plt.legend()`를 호출하여 그래프의 각 선이 무엇을 의미하는지 나타내는 범례를 추가합니다.

*   **`show_history` 함수 코드:**
    ```python
    import matplotlib.pyplot as plt

    def show_history(history, metric='acc'):
        """
        딥러닝 모델의 학습 이력(손실 또는 정확도)을 그래프로 시각화합니다.

        Args:
            history (dict): 모델 학습 이력을 담은 딕셔너리.
                            예: {'train_loss': [...], 'train_acc': [...],
                                 'val_loss': [...], 'val_acc': [...]}
            metric (str): 시각화할 지표. 'loss' 또는 'acc' (기본값: 'acc').
        """
        if metric == 'loss':
            train_metric_name = 'train_loss'
            val_metric_name = 'val_loss'
        else: # metric == 'acc'
            train_metric_name = 'train_acc'
            val_metric_name = 'val_acc'

        # 훈련 데이터 플로팅
        plt.plot(history[train_metric_name], label='train')
        # 검증 데이터 플로팅
        plt.plot(history[val_metric_name], label='valid')
        # 범례 표시
        plt.legend()
        # 그래프 제목 및 축 라벨 설정 (필요시 추가)
        plt.title(f'{metric.capitalize()} over Epochs')
        plt.xlabel('Epochs')
        plt.ylabel(metric.capitalize())
        # 그래프 보여주기 (Colab 환경에서는 보통 plt.show() 생략 가능)
        plt.show()
    ```

**3. 모델 학습 이력 시각화의 중요성**
*   **과적합(Overfitting) 감지:** 훈련 손실은 계속 감소하고 훈련 정확도는 계속 증가하지만, 검증 손실이 증가하거나 검증 정확도가 정체/감소하는 경향이 보이면 과적합을 의심할 수 있습니다. 시각화를 통해 이러한 현상을 조기에 발견하고 모델 개선에 활용할 수 있습니다.
*   **모델 성능 추이 분석:** 에포크가 진행됨에 따라 모델의 학습이 얼마나 잘 이루어지고 있는지(손실 감소, 정확도 증가)를 한눈에 파악할 수 있습니다.
*   **하이퍼파라미터 튜닝 보조:** 다양한 하이퍼파라미터(학습률, 배치 크기, 에포크 수 등) 설정에 따른 모델 학습 성능 변화를 시각적으로 비교하여 최적의 하이퍼파라미터를 찾는 데 도움을 줍니다.
