## 1. 🌟 강의 핵심 요약
1. 신경망의 모든 뉴런에는 **비선형적인 특성**을 학습에 반영하기 위해 반드시 **활성화 함수(Activation Function)**를 적용해야 합니다.
2. 활성화 함수는 은닉층(Hidden Layer)뿐만 아니라 **마지막 출력층(Output Layer)**에서도 결과값의 성격에 맞춰 적절하게 사용됩니다.
3. 은닉층에서는 주로 **ReLU**가 사용되며, 출력층에서는 이진 분류 시 **Sigmoid**, 다중 분류 시 **Softmax**를 사용하는 것이 일반적입니다.

## 3. 💡 핵심 개념 및 공식 정리

### ① 활성화 함수의 필요성
신경망에서 가중치 합($\Sigma$)에 비선형 함수를 적용하지 않으면, 층을 아무리 깊게 쌓아도 결국 하나의 선형 함수로 표현됩니다. 따라서 **비선형적인 부분(Non-linear part)**을 학습하기 위해 활성화 함수가 필수적입니다.

### ② 주요 활성화 함수 상세 해설

*   **ReLU (Rectified Linear Unit)**
    *   **특징:** 은닉층(Hidden Layer)에서 **가장 많이 사용**되는 함수입니다.
    *   **장점:** 연산이 매우 빠르고, 기울기 소실(Vanishing Gradient) 문제를 완화해 줍니다.
    *   **변형:** Leaky ReLU, ELU 등이 존재합니다.

*   **Sigmoid**
    *   **특징:** 출력값을 0과 1 사이로 압축합니다.
    *   **용도:** **이진 분류(Binary Classification)**의 마지막 출력층에서 Logit 값을 확률값으로 변환할 때 주로 사용됩니다. (최근 은닉층에서는 잘 사용되지 않음)

*   **Softmax**
    *   **특징:** 출력값들의 총합이 1이 되도록 만듭니다.
    *   **용도:** **다중 분류(Multi-class Classification)**의 마지막 출력층에서 사용됩니다.

### ③ 관련 공식
뉴런의 출력값 $Y$는 다음과 같이 계산됩니다.
$$Y = f(\sum_{i=1}^{n} w_i x_i + b)$$
- $x$: 입력값
- $w$: 가중치(Weight)
- $b$: 편향(Bias)
- $f$: **활성화 함수 (Activation Function)**

## 4. ✍️ 학습 퀴즈

**Q1. 신경망의 모든 뉴런에 활성화 함수를 적용해야 하는 근본적인 이유는 무엇인가요?**
> **정답:** 모델이 데이터의 비선형적인 특성(Non-linear mapping)을 학습하고 복잡한 문제를 해결할 수 있도록 하기 위해서입니다.

**Q2. 최근 딥러닝 모델의 은닉층(Hidden Layer)에서 Sigmoid보다 ReLU를 더 선호하는 이유는 무엇인가요?**
> **정답:** ReLU는 연산이 빠르고, 층이 깊어질 때 발생하는 기울기 소실 문제를 해결하는 데 훨씬 유리하기 때문입니다.

**Q3. 3개 이상의 클래스를 분류하는 '다중 분류' 문제에서 마지막 출력층에 가장 적합한 활성화 함수는 무엇인가요?**
> **정답:** Softmax 함수입니다. (각 클래스에 속할 확률의 총합을 1로 만들어주기 때문입니다.)
