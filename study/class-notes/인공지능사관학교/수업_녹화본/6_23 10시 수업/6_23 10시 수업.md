### 3. 핵심 개념 및 코드/공식 정리

#### 1. Batch Normalization (배치 정규화)
*   **목적**:
    *   **내부 공변량 변화(Internal Covariate Shift, ICS) 완화**: 신경망의 각 레이어를 통과할 때마다 입력 데이터의 분포가 바뀌는 현상을 줄여 학습을 안정화합니다.
    *   **기울기 소실(Vanishing Gradient) 및 폭주(Exploding Gradient) 문제 완화**: 활성화 값의 분포를 일정하게 유지하여 기울기가 너무 작아지거나 커지는 것을 방지합니다.
    *   **학습 속도 향상**: 안정적인 분포 덕분에 더 높은 학습률을 사용할 수 있어 수렴 속도가 빨라집니다.
    *   **정규화(Regularization) 효과**: 약간의 노이즈를 주입하는 효과가 있어 드롭아웃(Dropout) 없이도 과적합을 방지하는 데 도움을 줍니다.
*   **작동 방식**:
    *   미니 배치 단위로 각 특성(채널)의 평균과 분산을 계산합니다.
    *   이 평균과 분산을 이용해 활성화 값을 정규화합니다.
    *   정규화된 값에 스케일(scale)(`gamma`, `γ`)과 시프트(shift)(`beta`, `β`) 파라미터를 적용하여 학습 가능하게 만듭니다. 이 파라미터들은 BN이 너무 엄격하게 정규화하여 표현력을 잃는 것을 방지합니다.
*   **적용 위치**: 일반적으로 `Convolutional Layer` 또는 `Linear Layer` 이후, `Activation Function` 이전에 적용됩니다.
*   **PyTorch 구현**: `nn.BatchNorm2d(num_features)`
    *   `num_features`: 이전 레이어의 출력 채널 수(컨볼루션 레이어의 `out_channels`)와 동일하게 설정합니다.

#### 2. `nn.Sequential`
*   **목적**: 여러 개의 모듈(레이어)을 순차적으로 연결하여 하나의 큰 모듈(블록)로 정의할 수 있는 컨테이너 클래스입니다.
*   **장점**:
    *   **코드 간결화**: 복잡한 아키텍처를 논리적인 블록으로 묶어 코드를 더 읽기 쉽고 관리하기 쉽게 만듭니다.
    *   **모듈화**: 반복되는 레이어 패턴을 하나의 블록으로 묶어 재사용성을 높입니다.
    *   **유연성(가지치기)**: 모듈로 정의된 블록은 독립적인 객체처럼 다룰 수 있어, 특정 블록을 쉽게 교체하거나 수정하여 다양한 아키텍처를 실험할 수 있습니다. (강사님이 강조한 "가지치기" 개념)

#### 3. `nn.AdaptiveAvgPool2d(output_size=(1,1))`
*   **목적**: 입력 텐서의 공간 차원(높이, 너비)을 지정된 `output_size`로 평균 풀링하여 줄입니다. 글로벌 평균 풀링에 자주 사용됩니다.
*   **장점**: 입력 이미지의 크기에 관계없이 다음 레이어로 전달되는 특성 맵의 크기를 고정(예: 1x1)시켜, 모델을 더 견고하게 만듭니다.

#### 4. `torch.flatten(input, start_dim=1)`
*   **목적**: 입력 텐서를 1차원으로 펼칩니다. `start_dim=1`은 배치 차원(첫 번째 차원)을 제외하고 나머지 차원들을 펼치라는 의미입니다.
*   **용도**: 컨볼루션 레이어와 풀링 레이어를 거친 후, Fully Connected(선형) 레이어의 입력으로 사용하기 위해 데이터를 1차원 벡터 형태로 변환할 때 사용됩니다.

#### 5. Batch Normalization 적용 코드 예시 (수정된 `SimpleCNN02` 클래스 내부)
```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class SimpleCNN02(nn.Module):
    def __init__(self, num_classes=10, NUM_INPUT_CHANNELS=3):
        super().__init__()

        # padding=1로 Conv 적용 후 출력 면적 사이즈를 입력 면적 사이즈와 동일하게 유지.
        # kernel 크기 3, filter 개수 32 연속 적용.
        self.conv_block_1 = nn.Sequential(
            nn.Conv2d(in_channels=NUM_INPUT_CHANNELS, out_channels=32, kernel_size=3, padding='same'),
            nn.BatchNorm2d(num_features=32), # Batch Normalization 추가
            nn.ReLU(),
            nn.Conv2d(in_channels=32, out_channels=32, kernel_size=3, padding='same'),
            nn.BatchNorm2d(num_features=32), # Batch Normalization 추가
            nn.ReLU(),
            nn.MaxPool2d(kernel_size=2)
        )
        # out_channels이 64인 2개의 Conv2d 연속 적용. stride=1이 기본값, padding='same'은 version 1.8에서 소개됨.
        self.conv_block_2 = nn.Sequential(
            nn.Conv2d(in_channels=32, out_channels=64, kernel_size=3, padding='same'),
            nn.BatchNorm2d(num_features=64), # Batch Normalization 추가
            nn.ReLU(),
            nn.Conv2d(in_channels=64, out_channels=64, kernel_size=3, padding='same'),
            nn.BatchNorm2d(num_features=64), # Batch Normalization 추가
            nn.ReLU(),
            nn.MaxPool2d(kernel_size=2)
        )
        # Sequential Module을 이용하여 Conv Layer들을 생성. 이 경우 relu activation위해 ReLU Layer 연결 생성 필요.
        # filter개수 128개인 Conv Layer 2개 적용 후 Max Pooling 적용.
        self.conv_block_3 = nn.Sequential(
            nn.Conv2d(in_channels=64, out_channels=128, kernel_size=3, padding='same'),
            nn.BatchNorm2d(num_features=128), # Batch Normalization 추가
            nn.ReLU(),
            nn.Conv2d(in_channels=128, out_channels=128, kernel_size=3, padding='same'),
            nn.BatchNorm2d(num_features=128), # Batch Normalization 추가
            nn.ReLU(),
            nn.MaxPool2d(kernel_size=2)
        )

        # GAP 및 최종 Classifier Layer
        self.adapt_pool = nn.AdaptiveAvgPool2d(output_size=(1,1))
        self.classifier = nn.Linear(in_features=128, out_features=num_classes)

    def forward(self, x):
        x = self.conv_block_1(x)
        x = self.conv_block_2(x)
        x = self.conv_block_3(x)
        x = self.adapt_pool(x)
        x = torch.flatten(x, start_dim=1)
        x = self.classifier(x)
        return x
```
