### 3. 핵심 개념 및 코드/공식 정리

#### 1. 모듈화 (Modularity) 및 `nn.Sequential`
- **개념**: PyTorch에서 신경망 계층들을 그룹화하여 재사용 가능한 블록으로 만드는 기법. 코드의 가독성, 관리 용이성, 재사용성을 높여줍니다.
- **`nn.Sequential`**: 여러 모듈을 순차적으로 연결하여 하나의 큰 모듈처럼 작동하게 합니다.
- **예시 코드 (`self.classifier_block` 정의):**
    ```python
    self.classifier_block = nn.Sequential(
        nn.AdaptiveAvgPool2d(output_size=(1,1)),
        nn.Flatten(),
        nn.Linear(in_features=128, out_features=num_classes)
    )
    ```

#### 2. `nn.AdaptiveAvgPool2d`
- **개념**: 적응형 평균 풀링. 입력 텐서의 공간(height, width) 차원을 지정된 `output_size`로 줄입니다. 주로 Convolutional Layer 다음에 Flattening 직전에 사용하여 고정된 크기의 특징 맵을 생성할 때 사용됩니다.
- **예시**: `nn.AdaptiveAvgPool2d(output_size=(1,1))`는 각 채널에 대해 1x1 크기의 평균 풀링을 수행합니다 (Global Average Pooling).

#### 3. `nn.Flatten()`
- **개념**: 다차원 텐서(예: `(batch_size, channels, height, width)`)를 1차원 벡터(예: `(batch_size, channels * height * width)`)로 변환합니다. 이는 완전 연결 계층(Linear Layer)의 입력 형식에 맞추기 위해 필수적입니다.

#### 4. `nn.Linear` (Fully Connected Layer)
- **개념**: 입력과 출력을 연결하는 완전 연결 선형 변환 레이어. `in_features`는 입력 벡터의 크기, `out_features`는 출력 벡터의 크기를 지정합니다.

#### 5. Dropout (`nn.Dropout(p)`)
- **개념**: 신경망의 과적합(Overfitting)을 방지하기 위한 정규화(Regularization) 기법입니다. 훈련 중 무작위로 일정 비율(`p`)의 뉴런을 비활성화(출력을 0으로 설정)하여, 모델이 특정 뉴런에 과도하게 의존하는 것을 막고 더 강건한 특징을 학습하도록 유도합니다.
- **`p`**: 뉴런을 비활성화할 확률 (0.0에서 1.0 사이).
- **중요 특징**: **오직 훈련(Training) 시에만 적용**되며, 추론(Inference) 또는 평가(Evaluation) 시에는 적용되지 않습니다.
- **예시**: `nn.Dropout(p=0.5)` (50% 확률로 뉴런 비활성화)

#### 6. 모델의 동적 수정 (Dynamic Model Modification)
- **개념**: 이미 생성된 PyTorch 모델 인스턴스의 특정 서브 모듈을 런타임에 다른 모듈로 교체하는 기능. 이는 다양한 아키텍처를 실험하거나, 사전 훈련된 모델의 특정 부분을 미세 조정할 때 매우 유용합니다.
- **시연 코드 (`do_classifier_block` 정의 및 교체):**
    ```python
    # 새로운 분류 블록 정의 (Dropout 포함)
    do_classifier_block = nn.Sequential(
        nn.AdaptiveAvgPool2d(output_size=(1,1)),
        nn.Flatten(),
        nn.Dropout(p=0.5), # 첫 번째 Dropout
        nn.Linear(in_features=128, out_features=64),
        nn.ReLU(),
        nn.Dropout(p=0.3), # 두 번째 Dropout
        nn.Linear(in_features=64, out_features=10)
    )

    # SimpleCNN 모델 인스턴스 생성
    simple_cnn = SimpleCNN(num_classes=10)

    # 기존 classifier_block을 새로 정의한 do_classifier_block으로 교체
    simple_cnn.classifier_block = do_classifier_block
    ```

#### 7. `nn.ReLU()` (Rectified Linear Unit)
- **개념**: 활성화 함수 중 하나로, 입력이 0보다 작으면 0을 출력하고, 0보다 크면 입력을 그대로 출력합니다. 신경망에 비선형성을 추가하여 복잡한 패턴 학습을 가능하게 합니다.
