# Jugraid Routine (확정판)

## 기본 원칙
- 하루 1끼만 먹기 (점심 12:00, 집에서)
- 크레아틴: 집 들릴 때마다 3g씩 (아침/점심/저녁)
- 운동 후 집에서: 마그네슘(영양제) + 프로틴 60g 같이 섭취
- 물: 1시간에 600ml 이상
- 차전자피: 식사 50분 전

## 하루 타임라인

```
08:00 기상 + 세수 + 머리감기
08:05 비타민C / 철분 / 유산균 / 크레아틴 3g
08:10 선크림, 짐 챙기기
08:30 집 출발 (준비 30분)
09:05 학교 도착 (35분 이동)

[오전 수업]

11:50 집 복귀
11:50 크레아틴 3g
11:55 차전자피
12:00 점심 (파스타 1.5인분 + 닭다리살 200g)
12:30 커피 1잔

[오후 수업]

18:00–20:40 복습
20:40–20:45 저녁 체크인
20:45 학교 → 집 복귀
20:45–21:00 옷 갈아입기 + 프로틴 60g 준비(운동 가져감)
21:00 조선대 출발
21:00–21:30 농구
21:30–22:00 러닝
22:00 귀가
22:00 귀가 + 크레아틴 3g
22:00 마그네슘(영양제) + 프로틴 60g 섭취
22:00–22:25 팔굽혀펴기 실패할 때까지
22:25–22:40 샤워
22:40 취침
```

## 참고
- 프로틴은 러닝 끝나고 오는 길에 마심 = 운동 직후 효과 유지
- 크레아틴은 하루 9g (3g × 3회)
- 식사는 1일 1끼, 1끼만 먹음
- 첫 주에는 크레아틴 10g 정도 목표 (현재 빠지면 9g)
- 팔굽혀펴기는 실패할 때까지 = 근력 한계까지
